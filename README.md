### 🔐 &nbsp;Centralized Identity Provider
OAuth 2.0 + OIDC compliant IdP serving **5 products** with:
- JWT with **EdDSA** signing
- PKCE for public clients
- Refresh token families with cascade revocation
- Per-product token isolation & subscription-based auth
- **29 endpoints** across Auth, Session, OIDC, Audit routers

</td>
<td width="50%" valign="top">

### 📊 &nbsp;ETL Pipeline Architecture
End-to-end data processing with:
- **Airflow** orchestration + **PySpark** transformations
- Idempotent pipeline design — safe re-execution guaranteed
- **PostgreSQL + pgvector** for semantic search
- Processing **200K+ recruitment records**
- Schema-inconsistency handling across data sources

</td>
</tr>
<tr>
<td width="50%" valign="top">

### 🕷️ &nbsp;Distributed Web Scraping
High-throughput scraping infrastructure:
- Source-specific worker pools (LinkedIn, Naukri)
- **Redis/Celery** task distribution
- Planned **Kubernetes** migration with KEDA autoscaling
- Rate limiting & anti-detection strategies

</td>
<td width="50%" valign="top">

### 🔍 &nbsp;Open Source — Typesense
Diagnosed critical performance issue ([#2638](https://github.com/typesense/typesense/issues/2638)):
- **Raft consensus** thread starvation during 228K bulk inserts
- Write queue blocking `/health` & `/search` endpoints
- Proposed decoupled health endpoints with write backpressure
- Systems-level debugging at the consensus protocol layer

</td>
